Комментарии для запуска:
в запуске в дебагере в силионе в run -> edit configuration -> working directory -> src нужно щелкнуть на папочку и выбрать папку src, чтобы можно было бы найти файл из папки
в случае плохих исходов аффиных преобразований это должно не ломать программу, а просто не будет происходить перезаписи матрицы

Комментарии по проекту:
в таблице координат вершин - первая строчка не будет использоваться, так как для прорисовки отсчет вершин начинается с 1. В таблице прорисовок - все, как всегда
история без вершин и/или полигонов будет восприниматься как ошибочная
в масштабировании если не изменяется координата, то передавать надо 1, а не 0
сейчас у меня поле [-1;1]
в тестах прорисовки центра максимумы и минимумы должны быть зеркальны
некоторые free в тестах нужны для того, что если в тесте что-то пойдет не так, все не закрашилось с сегфолтом

по struct Coord_matrix:
rows - так как в f вершины начинаются с 1, то row будет number_vertex + 1 и нулевая строчка будет пустая
cols - always 3: x, y and z
